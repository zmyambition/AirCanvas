## 📖 项目介绍

**空中自由画板**是一个基于计算机视觉与手势识别的智能交互式绘画系统。用户无需借助屏幕、数位板或任何物理画笔，仅通过识别特定的手势动作，即可在空中控制虚拟画笔进行创作。

系统深度融合了高精度实时手势追踪、单目视觉深度感知、动态智能笔刷模拟、全色域 HSV 调色板以及独创的手势触发保存机制等核心技术，构建了一个兼具高自由度与强表现力的沉浸式绘画环境，提供了自然、直观且富有表现力的创作体验。

本项目旨在探索新一代自然人机交互范式，通过技术手段将艺术创作从物理工具的束缚中解放出来，并将其与计算机视觉的前沿实践深度结合。它不仅是一个技术演示，更是一个完整的创作工具，在教育展示、数字艺术创作、互动娱乐及交互装置艺术等领域均具有明确的应用潜力与实用价值。

## ✨ 功能实现

### 1. 核心交互系统 🖐️
*   **双模式智能切换**：
    *   **绘画模式**：仅竖起食指，在空中自由绘制线条。
    *   **悬停/选色模式**：食指 + 中指同时竖起，移动手指不产生笔迹，用于精确选择颜色与笔刷。
*   **智能防抖技术**：
    *   采用指数移动平均（EMA）算法。
    *   实时消除手部自然颤抖，确保线条流畅顺滑。

### 2. 多样化笔刷系统 🖌️
*   **Pen（钢笔）**：标准实心线条，适合轮廓勾画与精细描绘。
*   **Brush（毛笔）**：采用双重动态控制技术。
*   **Pencil（铅笔）**：半透明细线笔触，模拟素描草稿的自然质感。
*   **Neon（荧光笔）**：自带发光光晕特效，在黑底画布上呈现霓虹灯般的视觉效果。

### 3. 全色域调色系统 🌈
*   **无限色彩选择**：顶部集成 1000 像素宽彩虹色带，支持数百万种 BGR 颜色自由选取。
*   **实时视觉反馈**：右下角动态预览圆圈，同步显示当前选中颜色与笔触粗细。
*   **智能橡皮擦**：右上角专属功能区，根据画布背景色自动切换擦除模式。

### 4. 画笔粗细阶梯化调节 📏
*   为了解决空手绘图中因手部细微抖动导致的画笔参数不稳定问题，本项目引入了0.1 步进式的阶梯化调节机制。用户通过左侧 UI 滑块调整画笔粗细时，数值会像“磁吸”一样自动锁定在最近的档位上（例如 0.5x, 0.6x...1.5x），而非不稳定的连续浮点数。
    *   抗抖动设计：通过将连续的物理坐标映射为离散的档位区间，有效过滤了传感器捕获到的细微手部震颤。
    *   交互确定性：为用户提供明确的反馈，确保每次调整都能稳定停留在预期的数值，提升了操作的“确定感”。
    *   视觉反馈同步：UI 滑块的物理位置与数值显示同步进行阶梯化跳动，实现了视觉与逻辑的高度统一。

### 5. 智能化手势操作 👋
*   **五指张开清屏**：完全张开五指触发清屏功能，保持 1 秒后自动清空画布（含防误触设计），提升操作仪式感。
*   **“OK”手势保存**：双手比出“OK”手势，触发瞬间屏幕闪烁白光模拟闪光灯，显示“ART SAVED!”提示，作品自动保存为 PNG 格式至 output 文件夹。

### 6. 画布管理与系统功能 🖼️
*   **一键切换背景**：伸出食指、中指、无名指，支持黑白画布即时切换（切换时自动清屏），保持视觉一致性。
*   **一键撤销操作**：握拳即可撤销上一笔绘制，防止误操作，提升容错性。
*   **双窗口显示模式**：
    *   **主窗口（Main View）**：摄像画面与画布实时融合，便于手部定位与创作对齐。
    *   **纯净窗口（Pure Canvas）**：仅显示最终绘画作品，去除背景干扰，专注艺术呈现。

## 🛠️ 技术栈

本项目基于 Python 开发，主要依赖以下核心库：

| 库名称 | 用途 |
| :--- | :--- |
| **OpenCV (cv2)** | 图像处理、视频流捕获、图像融合、UI 绘制 |
| **MediaPipe** | 高精度手部关键点检测 (Hand Tracking) |
| **NumPy** | 矩阵运算、画布创建、数学计算 (插值、距离等) |
| **OS / Time / Math** | 系统路径管理、时间控制、几何运算 |

## 🚀 快速开始

只需简单三步，即可开启你的空中绘画之旅：

### 1. 克隆项目到本地
打开终端（Terminal）或命令提示符（CMD），运行以下命令将项目下载到本地：
```bash
git clone https://github.com/zmyambition/AirCanvas.git
cd AirCanvas
```

### 2. 安装环境依赖
确保你的电脑上已经安装了 Python (建议版本 3.8+)，然后运行以下命令一键安装所需的第三方库（OpenCV, MediaPipe, NumPy）：
```bash
pip install -r requirements.txt
```

### 3. 启动程序
```bash
python main.py
```
